

The user is asking how to update their GitHub repository with the latest changes. This is a knowledge question, not an implementation task — no plan needed.

Lovable has a two-way sync with GitHub. Changes made in Lovable automatically push to the connected GitHub repository. If the repo isn't updating, the most likely causes are:

1. **GitHub isn't connected yet** — The user needs to go to **Settings → GitHub → Connect project** to link a repo.
2. **Sync is stuck** — Sometimes disconnecting and reconnecting resolves it.
